# FRONTEND

## Технические требования

* **npm**

## Установка

1. В файле `frontend/constants.js` в переменной `api_url` должен быть записать адресс
1. Из папки `frontend` выполните:
  ```
  npm install
  ```
## Развёртка
1. Из папки `frontend` выполните:
  ```
  npm run
  ```
2. Перейдите по адресу http::/localhost:3000/
